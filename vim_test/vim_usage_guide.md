# Vim 사용 가이드

## 1. Vim 소개

Vim은 **터미널 기반 텍스트 에디터**로,  
코드 편집, 설정 파일 수정, 서버 환경 작업 등에 널리 사용됩니다.

- `vi`를 개선한 에디터: **Vi IMproved = Vim**
- **모드 기반**(입력 모드 / 일반 모드 / 명령 모드)이라는 점이 가장 큰 특징

## 2. 설치하기

### 2.1 Linux에서 설치

배포판에 따라 패키지 매니저로 설치합니다.

#### Debian / Ubuntu 계열

```bash
sudo apt update
sudo apt install vim
```

#### Fedora / CentOS / RHEL 계열

```bash
sudo dnf install vim
# 또는
sudo yum install vim
```

#### Arch / Manjaro 계열

```bash
sudo pacman -S vim
```

설치 확인:

```bash
vim --version
```

버전 정보가 출력되면 정상 설치된 것입니다.

### 2.2 macOS에서 설치

macOS에는 **기본적으로 Vim이 설치되어 있습니다.**

터미널에서 다음 명령으로 확인할 수 있습니다.

```bash
vim --version
```

버전 정보가 출력되면 기본으로 탑재된 Vim입니다.

추가로, 최신 버전 Vim을 쓰고 싶다면 Homebrew를 통해 설치할 수 있습니다.

```bash
brew install vim
```

설치 후:

```bash
which vim
vim --version
```

으로 경로와 버전을 확인할 수 있습니다.


## 3. Vim 기본 사용법

### 3.1 Vim 실행 및 파일 열기

```bash
vim 파일명
```

예:

```bash
vim memo.txt
```

* 파일이 있으면 → 해당 파일 열기
* 파일이 없으면 → 새 파일을 만들 준비 상태로 열기

그냥 `vim`만 실행하면 빈 버퍼로 시작합니다.

### 3.2 Vim의 세 가지 모드

Vim을 이해하는 핵심은 **모드**입니다.

1. **일반 모드 (Normal mode)**

   * Vim 실행 후 기본 상태
   * **이동, 삭제, 복사, 붙여넣기, 명령 실행**을 하는 모드

2. **입력 모드 (Insert mode)**

   * 실제로 **텍스트를 입력하는 모드**

3. **명령 모드 (Command-line mode)**

   * `:`를 눌러 진입
   * `:w`, `:q`, `:wq` 등 명령어를 입력하는 모드

#### 모드 전환 요약

* 일반 → 입력

  * `i` : 커서 위치에서 입력 시작
  * `a` : 커서 다음 위치에서 입력 시작
  * `o` : 아래 줄에 새 줄 추가 후 입력
  * `O` : 위 줄에 새 줄 추가 후 입력

* 입력 → 일반

  * `Esc` 키

* 일반 → 명령

  * `:` 입력

### 3.3 저장 및 종료

1. **일반 모드인지 확인**

   * `Esc`를 한 번 눌러 모드를 정리합니다.

2. 아래 명령들을 사용합니다:

| 동작          | 명령 (`:` 입력 후 엔터) |
| ----------- | ---------------- |
| 저장          | `:w`             |
| 종료 (변경 없음)  | `:q`             |
| 저장 후 종료     | `:wq`            |
| 저장 없이 강제 종료 | `:q!`            |

#### 파일 이름이 없는 경우 (E32: No file name)

`E32: No file name` 오류가 뜬다면,
현재 버퍼가 어떤 파일에도 연결되어 있지 않다는 뜻입니다.
이럴 때는 파일 이름을 지정해서 저장해야 합니다:

```vim
:w 파일이름.txt
```

예:

```vim
:w memo.txt
```

### 3.4 커서 이동

모든 이동은 **일반 모드에서** 수행합니다. (`Esc` 후 사용)

#### 기본 이동

* `h` : 왼쪽
* `j` : 아래
* `k` : 위
* `l` : 오른쪽

#### 줄 단위

* `0` : 현재 줄 맨 앞으로
* `$` : 현재 줄 맨 뒤로

#### 단어 단위

* `w` : 다음 단어의 처음으로 이동
* `b` : 이전 단어의 처음으로 이동

#### 파일 기준

* `gg` : 파일 맨 위로 이동
* `G` : 파일 맨 아래로 이동


### 3.5 텍스트 입력

1. 일반 모드에서 `i`를 눌러 **입력 모드**로 진입
2. 텍스트 입력
3. 입력이 끝나면 `Esc`로 다시 **일반 모드**로 복귀

자주 쓰는 입력 명령:

* `i` : 현재 위치에서 입력
* `a` : 현재 글자 뒤에서 입력
* `o` : 아래 줄에 새 줄 추가 후 입력
* `O` : 위 줄에 새 줄 추가 후 입력

### 3.6 삭제, 복사, 붙여넣기

모두 **일반 모드**에서 사용합니다.

#### 문자/단어/줄 삭제

* `x` : 커서 위치의 문자 1개 삭제
* `dw` : 커서 위치부터 **단어 끝까지** 삭제
* `dd` : 현재 줄 전체 삭제

#### 복사 & 붙여넣기

* `yy` : 현재 줄 복사
* `p`  : 복사/삭제한 내용을 **커서 아래(또는 뒤)** 에 붙여넣기

예시:

1. `dd` → 현재 줄 잘라내기
2. `p` → 방금 삭제한 줄을 아래에 붙여넣기

### 3.7 실행 취소(Undo) 및 다시 실행(Redo)

* `u` : undo (실행 취소)
* `Ctrl + r` : redo (다시 실행)

여러 번 눌러 연속으로 되돌리거나 다시 실행할 수 있습니다.


### 3.8 검색

1. **일반 모드**에서 `/`를 입력
2. 찾고 싶은 문자열 입력
3. Enter

예:

```vim
/hello
```

검색 이동:

* `n` : 다음 검색 결과로 이동
* `N` : 이전 검색 결과로 이동

## 4. 연습 루틴 예시

아래 순서로 직접 연습해보면 금방 익숙해집니다.

1. 터미널에서:

   ```bash
   vim practice.txt
   ```

2. `i` → 텍스트 몇 줄 입력 → `Esc`

3. `:w` → 저장

4. `gg`, `G`, `w`, `b`, `0`, `$` 등을 눌러 커서 이동 연습

5. `dd`, `yy`, `p`, `u`, `Ctrl + r` 등 편집 명령 연습

6. `/`로 단어 검색 연습

7. 마지막에 `:wq`로 저장 후 종료

 

## 5. 마무리

이 문서는 Vim 입문자를 위해 **설치부터 기본 사용법**까지를 다루는 간단한 가이드입니다.
추가로 배우면 좋은 주제:

* 비주얼 모드 (`v`, `V`, `Ctrl + v`)
* 여러 줄 들여쓰기 (`>>`, `<<`)
* 설정 파일 (`~/.vimrc`) 활용
* 플러그인 관리 (vim-plug, dein 등)
